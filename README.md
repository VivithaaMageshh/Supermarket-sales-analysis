# 🛒 Optimizing Inventory Management on Supermarket Sales Data

## 📌 Project Overview
This project analyzes 1,000 supermarket sales transactions across three branches 
in Yangon, Naypyitaw, and Mandalay. The goal is to uncover sales trends, 
customer behaviour, and product line performance to support better inventory 
and business decisions.



## 📂 Dataset
- **File:** `SuperMarket Analysis.csv`
- **Records:** 1,000 transactions
- **Branches:** A, B, C across 3 cities
- **Features:** Invoice ID, Branch, City, Customer Type, Gender, 
  Product Line, Unit Price, Quantity, Tax, Total, Date, Time, 
  Payment Method, COGS, Gross Income, Rating



## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn




## 📊 Analysis Performed
- Data Cleaning & Missing Value Check
- Data Type Conversion
- Text Data Preprocessing
- Data Aggregation by City, Branch, Product Line
- Data Splitting (Train/Test)
- Normalization & Scaling
- Sales by City and Branch
- Product Line Performance
- Sales Trends Over Time
- Customer Behaviour Analysis



## 📈 Visualizations

- Bar Chart – Product Line Performance

- Bar Chart – Customer Behaviour (Gender & Payment)
